baxter_behaviors Module¶

Provides baxter specific behaviors to handle GUI callbacks. All implemented function here are callable from the menu. Otherwise you have to provide your class to the menu manager.

computeAndInsertBox(**kwargs)[source]¶

Computes the transformation for the 2 previously saved points in relation to the base and inserts the selected box type in the scene. This also includes additional transformations which are related to the box. Also writes everything to the current open task in case the transformation should be saved.

Parameters:side (str) – influences the name of the inserted box and broadcasted tf

Executes a saved trajectory.

The parameters below are not the method parameters but the entries in the kwargs
